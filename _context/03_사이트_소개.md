# 03. 사이트 소개 (CTA·정체성)

## 한 줄 소개
**핏기어(dietexercise7.com)** — 홈트 기구·운동 앱·웨어러블·운동 용품을 가격·가성비·공간·소음 기준으로 "찾아보는 운동 덕후 핏기어"가 솔직하게 비교·리뷰해주는 한국어 피트니스 리뷰 블로그.

## 다루는 카테고리
- 홈트 기구(덤벨·요가매트·밴드·폼롤러·턱걸이바·유산소 기구), 운동·식단 기록 앱, 웨어러블·스마트워치, 운동 용품의 추천·비교·가성비 글.

## 다루지 않는 것 (니치 가드레일)
- **비(非)YMYL 제품·앱 리뷰만.** "사는 물건/앱"을 리뷰한다.
- ⚠️ 의학·건강 조언(식단 효능·보충제 효과·질병/부상 치료) 금지. 식단 앱을 리뷰해도 "어떤 앱이 한식 기록에 편한가"이지 "무엇을 먹어야 살이 빠지나"가 아니다.

## 다른 피트니스 글과의 차별점
1. **내 상황에 맞춰 정리해준다** — 원룸/입문/예산/층간소음 기준으로 "뭐부터"를 순서로 제시.
2. **1인칭 '찾아본 덕후' 시점** — 광고 복붙이 아니라 스펙·후기를 직접 뒤져 비교한 과정.
3. **단점·주의점을 솔직하게** — 협찬 글이 잘 숨기는 유격·소음·층간소음·무료 제약을 짚는다.

## EEAT 신뢰 근거
- 가격·사양은 "정가 기준"·공식/판매 페이지 링크로 처리(자주 바뀌므로 단정 금지).
- 앱은 스토어 공개 정보(평점·무료 범위·가격) 기준임을 명시, 확인 시점("○○년 ○월 기준") 표기.
- 비교 기준(예: 한식 DB '강함/약함')은 판단 근거를 박스로 밝힌다.

## CTA 방침 (중요)
- **현재 쿠팡 제휴 미설정 → 구매·제휴 버튼 절대 금지.** 없는 제휴 링크·`.btn` 버튼 넣지 않는다.
- 가격·후기는 "판매 페이지에서 직접 확인하세요"로 안내하되 버튼은 만들지 않는다.
- 마무리 CTA는 "관련 글 더 보기"형 내부링크로만. 다음 글로 이어지게.
- ※ 제휴가 승인·설정된 뒤에만 `.btn`(rel="nofollow sponsored") 버튼을 추가한다(PUBLISH.md 규칙).
