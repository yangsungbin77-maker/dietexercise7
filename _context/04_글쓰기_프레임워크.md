# 04. 글쓰기 프레임워크 (글의 뼈대)

핏기어가 글을 쓸 때 따르는 구조 규칙. seo-blog-post 스킬·`automation/PUBLISH.md` 기준과 합쳐 적용.

## 도입 공식
- **1인칭 공감/문제 제기로 시작.** "비싼 기구부터 사고 두 달 뒤 빨래걸이 됐죠?"처럼 입문자 페인포인트를 핏기어의 시선으로 연다.
- 첫 문단(앞 100자 안)에 메인 키워드 1회 + "결론부터 말하면요" 식 선결론.

## 본문 구조
- **3줄 요약 박스**를 도입 직후 배치(`cg-box cg-summary`, 안쪽은 HTML 태그로).
- H2 7~10개로 큰 흐름. 각 H2는 독자 질문 하나에 답(고르는 기준 / 비교 / 무게·사양 / 주의점 / 가성비 등).
- 헷갈리거나 오해하기 쉬운 지점엔 **팁 박스**(`cg-box cg-tip`), 피해야 할 건 **주의 박스**(`cg-box cg-warn`).
- 비교·정리는 **표**로(방식별·예산별·앱별). "한눈에 정리" 표 1개 권장.
- **FAQ 섹션**(Q&A 4개 이상) 필수 — PAA(함께 묻는 질문) 정조준.

## 1인칭 변수 주입 (글당 4~6회)
- 도입: 헤맨/오해했던 경험 1개.
- 본문 중간: "스펙을 직접 비교해봤다" 또는 "후기를 찾아보니" 1~2개.
- 비교표·주의점 부근: 입문자 공감 1개.
- 자연스럽게, 억지로 매 문단 넣지 않는다. ⚠️ 안 써본 제품을 "써봤다"로 쓰지 않는다(01·06 가드레일).

## 키워드·분량
- 메인 키워드는 제목·첫 문단·H2 최소 1개·마무리에 분산(본문 5~7회). 과밀 금지, 동의어·연관어로 분산.
- 본문 최소 2,500자(권장 2,500~3,500), 상위 경쟁글 평균보다 충실하게. 표·박스·FAQ 포함.

## 링크 규칙
- **외부링크 1개 이상** — 공식/신뢰 출처(보건복지부·공식 스토어·제조사 등) 또는 카테고리 페이지. assignment의 `externalLink` 우선.
- **내부링크 1개 이상** — 같은 주제군 기존 글(`/home-workout-beginner-gear/`, `/adjustable-dumbbell-guide/`, `/diet-macro-ratio/` 등). 클러스터로 묶는다.

## 이미지 (2개 필수)
- 히어로 `heroImage`(frontmatter) + 본문 중간 이미지 1개(`../../assets/posts/<키워드-slug>.png`, alt에 타깃 키워드 포함).

## 마무리
- 핵심 1줄 재정리 + "이거 하나만 기억하면 돼요" 톤.
- 가격·후기는 판매 페이지 확인 안내, 내부링크로 다음 글 회유. **버튼·제휴 CTA 금지**(03번).
- 끝에 `**관련 키워드** — #해시태그`(8~12개) 줄.
