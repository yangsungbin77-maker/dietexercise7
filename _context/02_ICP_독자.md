# 02. ICP · 독자

## 핵심 독자 1줄
운동·다이어트를 시작하려는데, 어떤 기구·앱을 사야 할지 검색하면 "추천 10선"만 줄줄이 나와서 정작 내게 맞는 게 뭔지 못 고르는 한국 입문자.

## 들어오는 검색 의도
- 추천·비교 — "홈트 기구 추천", "조절식 덤벨 추천", "다이어트 식단 앱 비교".
- 고르는 법 — "요가매트 두께", "덤벨 무게 고르는 법", "무료 운동 앱".
- 가성비·예산 — "입문자 홈짐 비용", "가성비 운동기구", "공짜로 어디까지".
- 주의점 — "층간소음 운동기구", "원룸 홈트", "살 때 주의".

## 독자 페인포인트 5개
1. 추천 목록만 많고 "내 상황(원룸·예산·입문)에 맞는 게 뭔지"는 안 알려준다.
2. 광고 문구·협찬 글이라 진짜 단점·주의점을 못 믿겠다.
3. 가격·사양이 글마다 다르고 옛날 정보라 지금도 맞는지 모르겠다.
4. 비싼 기구 질렀다가 안 쓰고 방치한 경험이 있어 또 실패할까 두렵다.
5. 무료라더니 핵심 기능은 다 유료라서, "공짜로 어디까지 되는지"가 궁금하다.

## 독자가 결정 직전 궁금해하는 것
- "이거, 원룸/입문/내 예산에 맞아?" (내 상황 매칭)
- "그래서 뭐부터 사면 돼?" (우선순위·결론 욕구)
- "단점·주의점은 없어?" (솔직한 후기 욕구)
- "지금도 이 가격 맞아?" (최신성 의심)

## 읽고 나서 바라는 행동
- 제휴(쿠팡파트너스) 미설정 상태 → 지금은 **버튼·구매 유도 금지**.
- 목표는 "내 상황에 뭘 고를지 정리됐다"는 만족 → 내부링크로 관련 글 회유 → 재방문.
- 가격·후기는 판매 페이지에서 직접 확인하도록 안내(단정 대신 "구매 전 확인").
